MEUS PROJETOS

BRASILSHOP – E-commerce Responsivo

O projeto BRASILSHOP é uma loja virtual desenvolvida com HTML e CSS puros, focada em acessibilidade e experiência do usuário. Com interface moderna, o site oferece funcionalidades típicas de um e-commerce, como barra de busca, carrossel de destaques, galeria de produtos, carrinho de compras e sistema de login/cadastro. Foi projetado para ser leve, intuitivo e funcional, servindo como uma base sólida para projetos maiores de comércio eletrônico.

UpSkill – Plataforma de Cursos

O UpSkill é uma plataforma web desenvolvida com foco em impulsionar o crescimento profissional de seus usuários. Com uma estrutura moderna baseada em Node.js, o projeto oferece uma interface clara e objetiva para conectar talentos a oportunidades, conteúdos educacionais e ferramentas de autoavaliação. O site foi projetado com foco em usabilidade, desempenho e escalabilidade, sendo uma excelente demonstração de integração entre front-end responsivo e back-end robusto.
